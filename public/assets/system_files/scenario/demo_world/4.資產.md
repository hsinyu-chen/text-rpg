# 資產

> **適用範圍**：本檔僅記錄**主角擁有的非隨身財產**——**寄存於據點／金庫的現金**、不動產與據點佈局。據點各房間下可列出**寄存於該地點的長期物資**。**主角口袋裡的隨身現金與物品**請寫至 `9.物品欄.md`（本檔的「動產 → 庫存現金」與物品欄的「隨身現金」是互斥的兩本帳）。**NPC 與同伴的私有資產**請寫至 `3.人物狀態.md` 該角色條目下，不寫於此。

```
        - **[房間名稱]**： [功能與內容；可列存放於此的長期物資]
```
