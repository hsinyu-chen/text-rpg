# 物品欄

> **適用範圍**：本檔僅記錄**主角擁有且隨身可動**之物（口袋、手上、背包、便攜空間等，是否屬於「隨身」由你依設定判斷），**包含主角口袋現金與隨身貴重小物**。**寄存於據點、旅館、他人保管處**等非隨身物資（含據點金庫的存款）請改寫至 `4.資產.md` 對應據點下；**NPC 與同伴的私人物品**請寫至 `3.人物狀態.md` 該角色的 `### 已知持有重要物品`。

## 存檔格式
```
## 【[存儲位置]】[容器名稱]
* **目前總質量**： 約 [數值] KG

### [分類標題，如：食材、裝備]

* [物品名稱] / [數量]
* [物品名稱] / [數量]
    - [備註說明]
```
