# Magic and Skills

## Magic Save Format
```
## [Magic Name]

* **Description**: [Magic Function, Principle Brief]
* **Execution Process**:
    1. [Step 1]
    2. [Step 2]
* **Effect**: [Physical or Magical Effect after casting]
* **Tactical Application**: [Application scenarios in combat or daily life]
```

## Skill Save Format
```
## [Skill Name]

* **Type**: [Sword/Spear/Martial/Special...]
* **Description**: [Skill effect summary]
* **Requirements**: [Prerequisites, costs, cooldowns...]
* **Tactical Application**: [Applicable scenarios]
```
