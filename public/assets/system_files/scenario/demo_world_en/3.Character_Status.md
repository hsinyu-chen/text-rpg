# Format Definition

- **Identity**: [Detailed identity description]
- **Basic Info**: [Race] / [Gender] / [Age] / [D&D Alignment]
- **Income**: [Source/Amount]
- **Last Known Location**: [Place] ([Calendar Date HH:MM; **date precise to day, clock time precise to minute**, e.g. `Saint Calendar 1000.04.02 Tuesday 18:40`])
- **Current Status**: [Physiological status only]
- **Current Mindset**: [Mindset description] (non-protagonist characters only)
- **Address for <!uc_name>**: [How they address the protagonist] (non-protagonist characters only)
- **View on <!uc_name>**: [Their opinion] (non-protagonist characters only)
- **Background**: [Background description]
- **Personal Details**: Appearance / Attire / Notes
