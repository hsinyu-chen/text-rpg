### 目前處理的檔案：人物狀態（角色卡、隊伍名冊、NPC 狀態）

輸出一份場景聚焦的角色名冊，**保留原始檔案的結構**（標題、順序、格式慣例）。對檔案中的每一位角色套用以下分級規則：

#### 甲級 — 完整保留（完整逐字複製該角色整個條目）

- **玩家角色**（主角／視角角色），一律甲級。
- 在「角色」欄位（**{{CHARACTER}}**）中列名的每一位角色。
- 與上述任一甲級角色具**密切關係**者 — 朋友、戀人、配偶、兄弟姊妹、父母／子女／其他家人、同學／同事、師長／弟子、直屬部下、上司、隊伍夥伴、值得注意的主角宿敵或對手。關係依來源檔案本身之描述判斷。

對於甲級角色：**禁止**改寫、精簡、或「摘要」其角色卡。數值、背景、特質、裝備行、怪癖、子項目 — 一切都必須保留。

#### 乙級 — 單行摘要（絕對不可完全刪除）

檔案中其餘所有角色都**必須保留**，但條目壓縮為**單一簡潔句**，並保留角色的名稱／識別。建議格式：

```
### 角色名 — 一句話描述其角色定位與目前狀態。
```

新章節仍需要能在這些人物被後續提及時辨認他們。

#### 其他規則

- 保留原始檔案的區段順序與結構慣例（例如「主要隊伍」、「盟友」、「敵人」等分類標題）。
- 不得將任何角色的狀態推進到開場場景之後。
