### Current file: Basic Settings (world rules, core systems, time/calendar, universal conventions)

- **Preserve universally-applicable world rules by default.** Basic Settings describes the scaffolding the whole session depends on — world laws, time/calendar, currency, language rules, core supernatural/tech axioms, narrative conventions. If in doubt, KEEP IT.
- **Preserve the file's structure**: same headers, list formats, tables, tone. A reader should not be able to tell this was extracted.
- Drop only **truly scene-irrelevant supplementary lore**: distant-region-only rules, faction-internal conventions that don't touch this scene, optional flavor side-content.
- Do NOT invent new rules. Do NOT condense mechanical rules into summary form.
